# GRILLAS_2000_9X9 🗄️

Este repositorio contiene el sistema de almacenamiento, generación y procesamiento masivo de grillas de Sudoku 9x9, desarrollado en Visual Basic 6.0.

## 🚀 Descripción

`GRILLAS_2000_9X9` es un proyecto robusto diseñado para gestionar grandes volúmenes de soluciones de Sudoku. El sistema permite la creación, indexación y consulta de miles de grillas, facilitando el análisis estadístico y la validación de plantillas (*templates*) a gran escala. 

Es una herramienta fundamental para la investigación de la distribución de soluciones y la búsqueda de patrones únicos en matrices de noveno orden.

## 🛠️ Especificaciones Técnicas

- **Lenguaje:** Visual Basic 6.0.
- **Arquitectura:** Orientada a la gestión de datos (Data Management) y procesamiento por lotes.
- **Funcionalidades Clave:**
  - Generación masiva de combinaciones de grillas 9x9.
  - Almacenamiento optimizado de soluciones para acceso rápido.
  - Interfaz de navegación para la exploración de la base de datos de grillas.
  - Filtros avanzados basados en propiedades matemáticas y códigos de plantilla.

## 📂 Estructura del Repositorio

Tras la reestructuración profesional:
- `/src`: Código fuente del gestor de grillas (.vbp, .bas, .frm).
- `/data`: (Opcional) Archivos de muestra o esquemas de base de datos.
- `/docs`: Documentación técnica sobre la metodología de generación y almacenamiento.
